# 🛍️쇼핑몰 만들기

- shop (api server)
- front (client server)
- API gateway 
- auth server (인증, 인가)

# Shop - Rest API Server
- 클라이언트 요청을 처리하는 api 관련 서버 입니다.

# Front - Client Server
- 클라이언트가 요청을 보낼 수 있는 서버로 이곳에서 front를 담당합니다.

# Api gateway
- 들어온 요청을 api gateway를 통해 처리할 수 있도록 하여 보안성을 올립니다.

# Auth Server
- front 서버에서 들어온 인증, 인가 관련 처리를 이곳 서버에서 처리합니다.
